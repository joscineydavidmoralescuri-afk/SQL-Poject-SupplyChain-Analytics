# SQL-Poject-SupplyChain-Analytics
Comparto mi primer proyecto final de análisis de datos en SQL Server (SSMS), aplicando herramientas de análisis profundo para el desarrollo de múltiples casuísticas como: Análisis de Pareto, Logística Predictiva, Crecimiento MoM, Promedios moviles, entre otros descubrimientos viendo de manera progresiva el nivel aplicado en cada caso.
